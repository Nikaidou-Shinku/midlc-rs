## midlc-rs

A simple MIDL compiler, made for coursework.

### Usage

```console
A simple MIDL compiler, made for coursework

Usage: midlc-rs [OPTIONS] <INPUT>

Arguments:
  <INPUT>  The MIDL source file

Options:
  -o <OUTPUT>      Place the output into <OUTPUT> [default: SyntaxOut.txt]
  -h, --help       Print help
  -V, --version    Print version
```
